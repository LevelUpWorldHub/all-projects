# Get It Done Roadmap

## Now

### RFQ/Bid/Change-Order Operating Loop

Objective:

Build the first commercial workflow:

```text
RFQ → Vendor Invite → Bid Submission → Bid Comparison → Award → Change Order → Approval + Audit Trail
```

Now items:

- RFQ/Bid happy path.
- Project and vendor foundations.
- RFQ creation.
- Vendor invite.
- Vendor RFQ view.
- Bid submission.
- Audit events.
- Permissions.
- Bid comparison.
- Award flow.
- Change-order creation.
- Approval flow.
- Edge-case test suite.

## Next

- Mobile field app.

## Roadmap Rules

1. Do not promote integrations to Now until the RFQ/Bid loop works end-to-end.
2. Do not add AI recommendation until structured bid data exists.
3. Do not add advanced analytics until there is enough workflow volume.
4. Do not add owner billing automation until change-order approval is reliable.
5. Do not expand beyond commercial workflow until the first loop has adoption and measurable value.
